### 心エコーの評価

#### 基準値

- 参: 小児・新生児診療ゴールデンハンドブック

|房室弁横径(mm)|12|15|17|20|23|29|32|36|

- 左室拡張末期短軸径(mm)
  - 身長 \< 75cm
    - 男児: 0.40 x Ht - 1.1
    - 女児: 0.39 x Ht - 3.0
  - 身長 ≧ 75cm
    - 男児: 0.22 x Ht + 12.2
    - 女児: 0.20 x Ht + 13.3
- 下大静脈系
  - 7.0±2.1 mm/m^2^
  - 11 mm/m^2^以上で循環血液量の過剰を疑う

#### 評価部位

- four-chamber view
  - 左室・右室・左房・右房のバランス
  - 心室中隔の形
- short axis view
  - FS(%) = (LVDd - LVDs) / LVDd x 100
    - Fractional Shortning
    - normal: 28-44%
  - EF(%) = (LVDd^3^ - LVDs^3^) / LVDd^3^ x 100
    - Ejection Fraction
    - 上記はPombo(Cube)法
            左室をラグビーボール上として計算、球体に近い拡張した左室だと値がずれる
    - normal: 64-83%
    - Gibson
      - EDV = π/6Dd^2^Ld, Ld = 0.98Dd + 5.9 -\> = Dd^3^(0.98Dd +
                5.90)
      - ESV = π/6Ds^2^Ls, Ls = 1.14Ds + 4.18 -\> =
                Ds^3^(1.14Ds + 4.18)
    - Tenchholz
      - V = π/6D^2^L, L = D/(0.075D + 0.18) -\> = 7.0 \* D^3^ /
                (2.4 + D)
    - D = 左室短形, L = 左室長径
