### 赤血球濃厚液(RCC-LR)

- 1u. 120mL
- 予想上昇Hb = (30 x 輸血単位(u.) ) / (0.75 x BW)
- 10mL/kgでHb 3g/dL上昇
- 2u. で約1Lの細胞外液不可に相当する
- 必要輸血量 [^1]

- 新生児 = (BW x 80 x (Ht期待値 - Ht実測値) ) / 投与血のHt
- 乳児以上 = (Ht期待値 -Ht実測値) x 2.2 x BW

[^1]: 小児ICUマニュアル
